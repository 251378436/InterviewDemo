# Introduction 

This repository contains Client/API/API.Tests, is used for demo

## Tooling

* git
* .NET 6.0 SDK
* Visual Studio, Visual Studio Code
* Angualr 13
* npm

## Run Client
cd ./src
use command: 
npm install
npm run start

## Run API
cd ./API
use command:
dotnet run

## Run API.Tests
cd ./API.Tests
use command:
dotnet test
